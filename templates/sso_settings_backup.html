Setup Instructions

To configure SSO providers, you need to create OAuth applications in their respective consoles:

  • Google: Google Cloud Console → APIs & Services → Credentials
  • Microsoft: Azure App Registration
  • Authentik: Applications → Providers → Create OAuth2/OpenID Provider
  • Keycloak: Clients → Create Client → OpenID Connect
